# Decimal to Binary Converter

This program converts a decimal number to a binary number


## Files

1. **main.cpp**: Handles user input and output.
2. **StackLinkedList.h**: Defines a stack-based linked list to handle the binary conversion of the integer part.
3. **QueueLinkedList.h**: Defines a queue-based linked list to handle the binary conversion of the fractional part.
4. **BinaryConverter.h**: Declares the BinaryConverter class, which performs the conversion.
5. **BinaryConverter.cpp**: Implements the BinaryConverter methods.
